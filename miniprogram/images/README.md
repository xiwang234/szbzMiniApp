# TabBar 图标说明

由于无法直接生成图片，请按照以下要求准备图标：

## 需要的图标文件

1. **tab-jitu.png** - 吉途未选中图标（灰色）
   - 尺寸：81px × 81px
   - 颜色：#999999
   - 建议图标：指南针、罗盘等

2. **tab-jitu-active.png** - 吉途选中图标（红色）
   - 尺寸：81px × 81px
   - 颜色：#C8102E
   - 建议图标：指南针、罗盘等

3. **tab-wenji.png** - 问吉未选中图标（灰色）
   - 尺寸：81px × 81px
   - 颜色：#999999
   - 建议图标：卦象、问号等

4. **tab-wenji-active.png** - 问吉选中图标（红色）
   - 尺寸：81px × 81px
   - 颜色：#C8102E
   - 建议图标：卦象、问号等

## 临时解决方案

如果暂时没有图标，可以：
1. 使用小程序开发工具的默认图标
2. 或者在 app.json 的 tabBar 配置中暂时注释掉 iconPath 和 selectedIconPath

## 图标制作工具

- 在线生成：https://www.iconfont.cn/
- 设计软件：Figma, Sketch, PS
- 简单工具：Canva, 创客贴
