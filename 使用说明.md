# 微信小程序使用说明

## 📂 项目结构说明

本项目现在包含两套代码：

### 1. 原生微信小程序（miniprogram 目录）✅ 推荐
```
miniprogram/
├── app.js          # 小程序主逻辑
├── app.json        # 小程序全局配置
├── app.wxss        # 小程序全局样式
├── sitemap.json    # 搜索配置
└── pages/          # 页面目录
    ├── index/      # 首页
    ├── info/       # 信息展示页
    ├── payment/    # 支付页
    └── result/     # 结果页
```

### 2. uni-app 多端框架（根目录）
- 需要编译后才能运行
- 支持多端部署（微信、抖音、H5、APP）

---

## 🚀 在微信开发者工具中打开

### 方法一：直接导入（推荐）

1. 打开**微信开发者工具**
2. 点击**导入项目**
3. 选择项目目录：`d:\project\szbzMiniApp`
4. AppID：选择**测试号**或填写您的 AppID
5. 点击**导入**

✅ **重要**：项目会自动识别 `project.config.json`，并读取 `miniprogram/` 作为小程序根目录

### 方法二：手动配置

1. 打开微信开发者工具
2. 创建新项目
3. 项目目录选择：`d:\project\szbzMiniApp`
4. AppID：测试号
5. 后端服务：不使用云服务
6. 模板：不使用模板

## 🔧 如遇到问题

### 问题1：找不到 app.json
**原因**：没有设置 miniprogramRoot  
**解决**：已在 `project.config.json` 中配置 `"miniprogramRoot": "miniprogram/"`

### 问题2：图片无法显示
**原因**：需要配置域名白名单  
**解决**：开发工具 → 详情 → 本地设置 → 勾选"不校验合法域名"

### 问题3：样式不生效
**原因**：外部样式表可能被拦截  
**解决**：已使用 Unicode 符号代替 Font Awesome 图标
